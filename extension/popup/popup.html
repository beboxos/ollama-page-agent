Ollama Page Agent

Configure la connexion a ton serveur Ollama local. Rien ne quitte ta machine.

Rafraichis la page apres avoir change ce reglage.

Vision (modeles multimodaux)
Envoie aussi une capture d'ecran a chaque etape (ex: gemma3, llava, qwen2-vl). Sans effet avec un modele texte seul.

Ollama ne repond pas ? (CORS)

Autorise cette extension a appeler ton serveur Ollama en definissant la variable d'environnement avant de lancer ollama serve :

OLLAMA_ORIGINS=chrome-extension://* ollama serve

Puis clique sur "Autoriser" ci-dessus et accepte la permission de navigation Chrome.
